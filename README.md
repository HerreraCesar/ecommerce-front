## ¿Como usar la aplicación de forma local? <a name="5"></a>

Este proyecto requiere **[Node.js](https://nodejs.org/)** para su ejecución.

```
git clone https://github.com/HerreraCesar/ecommerce-front.git
```
```
cd ecommerce-front
```
```
npm install
```
```
npm start
```
